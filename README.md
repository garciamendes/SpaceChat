### Installation
##### Requests
- Node JS > 14.x
- Npm > 7.x

##### Steps:
- clone the repository: ```git clone https://github.com/garciamendes/SpaceChat.git```
- In **frontend** folder run:
```npm install```

#### Running your project
- In frontend run: ```npm start``` or ```yarn start```
